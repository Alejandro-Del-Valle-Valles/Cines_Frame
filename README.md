# Cines_Frame
>Autores: Alejandro del Valle Vallés, Aurora Pinar del Hoyo, Carolina Lobato Ruiz

Repositorio del TFG de DAM 2 curso 2025-2026.

Este repositorio contiene el proyecto de la aplicación del TFG de DAM 2 del curso 2025-2026.

## Herramientas:
- Kotlin Multiplatform
- Compose Multiplatform
- Consumo de API hecha en Java con Spring

La aplicación funciona en sistemas de escritorio Windows y Linux, y sistemas móviles Android.
